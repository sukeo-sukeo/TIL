# グーグルアナリティクス設置フロー
```
1. 「管理」の「プロパティを新しく作成」
2.  必要事項を入力
3.  `<gtag>`をコピペして設置
4. 「すべてのwebページ」の「フィルタ」でipアドレスを指定して自分を除外
```
